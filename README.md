# Тестовое задание (Frontend)

Интерактивная таблица с графиком

# Демо
https://nadya-qo.github.io/test_delta/

## Репозиторий
https://github.com/nadya-qo/test_delta

## Файлы проекта
index.html — разметка страницы  
common.css — стили  
common.js — логика интерфейса  
data.json — данные для таблицы и графика  

## Описание

В тестовом задании реализуется отображение таблицы с данными и построение графика на основе JSON.
Используется chart.js для построения графика
